🚀 GitHub Pages Deployment Summary (With GitExtensions)

📁 Step 1: Prepare Your Project Folder

Organize your project like this:

sk8rd13/
├── index.html
└── assets/
    ├── logo.png
    └── background.mp3
🧩 Step 2: Initialize Repository with GitExtensions
  • Open GitExtensions
  • Go to Repository → Create new repository
  • Select your project folder
  • Click OK to initialize the repo
💾 Step 3: Stage and Commit Your Files
  • Select all files in the Working Directory Changes
  • Click Stage
  • Write a commit message like Initial commit
  • Click Commit
🌐 Step 4: Create GitHub Repository
  • Go to GitHub
  • Click New Repository
  • Name it (e.g. sk80rd13), make it Public or Private
  • Do not initialize with README or .gitignore
🔗 Step 5: Connect Remote in GitExtensions
  • Click Remote → Manage Remote Repositories
  • Name: origin
  • URL: https://github.com/yourusername/sk80rd13.git
  • Click Save changes, then Yes when prompted to auto-configure
📤 Step 6: Push to GitHub
  • Click Remote → Push
  • Make sure origin and main or master is selected
  • Click Push
🌍 Step 7: Enable GitHub Pages
  • Go to your repository on GitHub
  • Click Settings → Pages
  • Set Source to main branch, folder / (root)
  • Click Save
  • GitHub will give you a public URL like:
    https://yourusername.github.io/sk80rd13/
